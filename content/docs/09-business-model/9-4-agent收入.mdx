## Agent 收入的来源

Agent 收入可以来自配置、使用、执行和持续支持四个维度。

它既可以是对节点和项目的直接收入，也可以是嵌入服务和 Deal 中的间接收入。

## 为什么 Agent 收入成立

因为 Agent 不再只是‘帮你写点东西’，而是在 WCN 中承担正式执行角色。

只要它能进入任务、产出结果并被采纳，它就具备商业化基础。

<Callout type="warning">
Agent 的商业价值，不来自“模型有多强”，而来自“它是否进入了真实闭环”。
</Callout>

## Agent 收入和服务收入的关系

有些 Agent 收入可以独立存在，例如 Agent Desk 或 Agent Subscription。

也有一些 Agent 收入会嵌入到服务包和 Deal 收入中，成为隐藏在服务层中的自动化能力溢价。
