## 这一章回答的是：WCN 里的“人和组织”，到底以什么方式存在。

如果没有节点系统，WCN 就只是一个联系人网络；有了节点系统，它才成为一个责任网络、执行网络和结算网络。

## 节点系统的核心逻辑

节点只有在三个条件下才有意义：能带来资源、能推动业务、能接受规则。

<Callout type="warning">
节点不是“名字”，而是“位置”；不是“加入者”，而是“责任承担者”。
</Callout>
