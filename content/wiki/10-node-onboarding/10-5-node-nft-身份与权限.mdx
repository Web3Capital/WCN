## Node NFT 的角色

身份载体：证明某个节点在系统中的正式位置。

权限载体：映射访问权、参与权和部分未来系统能力。

记录载体：未来可映射节点等级、信誉与历史。

<CardGrid cols={2}>
  <Card title="不是投资产品" description="不能被写成保底收益或自动回报逻辑。" />

  <Card title="不是空壳收藏品" description="它的意义来自系统关系，而不是图像本身。" />

</CardGrid>

## 为什么 WCN 需要 NFT 形态

因为当网络成熟后，位置、权限、等级和长期记录都需要更标准化、更数字化的表达。

NFT 是一种可能的承载方式，但前提是业务网络先成立。

<Callout type="warning">
先有节点和闭环，再有 NFT 承载；不是先卖 NFT 再找意义。
</Callout>

## 当前阶段的边界

当前阶段不应把 Node NFT 表达为收益承诺或金融化产品。

当前更适合把它表达为：身份、权限、位置与未来长期数字化接口。
