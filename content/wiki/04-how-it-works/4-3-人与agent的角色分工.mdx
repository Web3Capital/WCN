## 人类节点负责方向、资源、信任和责任。

在 WCN 中，人类节点不是被 AI 替代的对象，而是整个系统的战略控制层。

<CardGrid>
  <Card title="负责资源输入">带项目、带资本、带服务、带地区资源——把机会真正带进系统。</Card>
  <Card title="负责判断与优先级">决定什么值得推进、什么需要暂停、什么该进入下一阶段。</Card>
  <Card title="负责关系与信任">高价值业务仍然需要人对人信任、判断与责任担保。</Card>
  <Card title="负责最终责任">一旦出现越权、错误或争议，责任不落在模型，而落在节点主体。</Card>
</CardGrid>

## Agent 负责执行、结构化和规模化放大。

Agent 的价值，不是“会聊天”，而是把本来依赖人工的大量流程性动作做得更快、更稳定、更可重复。

<Callout type="warning">
Agent 的核心角色，是把优秀节点的方法“流程化、结构化、可复制化”。
</Callout>

<CardGrid>
  <Card title="Research Agent">整理信息、生成摘要、做初步尽调、对比项目与风险。</Card>
  <Card title="Deal Agent">做初筛、匹配、纪要、候选推荐与线索整理。</Card>
  <Card title="Growth Agent">生成内容、跟踪归因、支持增长与传播执行。</Card>
  <Card title="Execution Agent">催办、提醒、收集回执、推进流程与里程碑。</Card>
</CardGrid>

## 什么必须留在人，什么可以交给 Agent？

<CardGrid>
  <Card title="必须留在人">签约、付款、法律税务结论、最终 PoB 批准、高风险营销、重大资源承诺。</Card>
  <Card title="可以交给 Agent">研究、筛选、跟进提醒、纪要、信息整理、状态跟踪、基础归因与监控。</Card>
  <Card title="必须有人类审批的动作">任何对外发送、关键材料确认、交易结构判断、敏感法域相关操作。</Card>
  <Card title="必须留下日志的动作">读取、推荐、触达、验证、标记、催办、输出等所有关键执行动作。</Card>
</CardGrid>

WCN 不追求让 Agent 全自动替代人，而追求让 Agent 成为受控、可审计、可结算的执行层。
