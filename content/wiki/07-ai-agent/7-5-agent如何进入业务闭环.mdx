## Agent 怎么进入闭环？

Agent 不是天然属于业务闭环，它必须通过正式工作流进入。

<Callout type="warning">
任务派发 → Agent 执行 → 输出被记录 → 人类节点采纳 → 输出进入结果链 → Proof Desk 审核 → PoB 记账
</Callout>

只有这样，Agent 的价值才不是抽象“有帮助”，而是可被系统识别的结果推动力。

## Agent 的输出怎样进入 Proof？

没有这些条件，Agent 的输出就只能算“工具生成物”，不能进入系统的长期价值层。

## 为什么这一页重要？

<Callout type="warning">
WCN 的关键不是“用了 AI”，而是“让 AI 第一次进入了可验证、可归因、可结算的业务网络”。
</Callout>
