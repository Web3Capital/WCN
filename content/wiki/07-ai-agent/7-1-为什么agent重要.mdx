## 为什么 WCN 不能只靠人？

因为高质量节点的时间是最稀缺资源。如果所有筛选、纪要、提醒、跟进和初步分析都只能由人完成，网络容量会很快触顶。

<Callout type="warning">
Agent 的价值，是把“高价值人类节点不该重复做的事”接下来。
</Callout>

## Agent 在 WCN 里能带来什么？

<CardGrid cols={2}>
  <Card title="提效" description="缩短研究、筛选、纪要和任务推进的时间。" />
  <Card title="标准化" description="让输出更有结构，更容易进入 Proof 和审核。" />
  <Card title="可审计" description="日志和版本记录让过程更可追踪。" />
  <Card title="可扩张" description="同一套能力可以被更多节点和区域复用。" />
</CardGrid>

## 为什么现在 Agent 特别重要？

因为今天 AI 的能力已经足够覆盖大量流程动作，但行业还没有把这些能力真正放进一个可验证、可结算的网络里。

WCN 的机会就在于：它不仅使用 Agent，而是给 Agent 一个正式位置。
