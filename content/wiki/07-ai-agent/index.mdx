  <Card title="7.3 Agent 生命周期" description="WCN 里的 Agent 不是“永久外挂”。它们必须像节点一样，经历注册、授权、运行、复核、升级、暂停和退出，才能形成可信执行层。" />
  <Card title="7.4 Agent 的边界" description="WCN 不是让 Agent 自由发挥，而是让它在清晰边界内最大化发挥。越靠近资金、法律、合同、治理和声誉，越需要把边界写死。" />
