<CardGrid cols={2}>
  <Card title="9.1 收入来源" description="一个真正可持续的系统，必须在 Token 之前就建立现金收入。WCN 的收入来源不是单点收费，而是围绕节点、Deal、服务和 Agent 形成多层收入结构。" />
  <Card title="9.2 节点席位模型" description="节点席位模型是 WCN 早期最重要的现金流之一。但它必须被正确表达：不是投资产品，不是自动回报，而是一个进入网络的正式位置。" />
  <Card title="9.3 服务与成交收入" description="WCN 不只是卖位置，它还提供真正能推进闭环的服务与 Deal 协作能力。因此，服务收入和成功费，是其最直接的商业验证。" />
  <Card title="9.4 Agent 收入" description="如果 Agent 只是免费工具，那它的价值很难被系统长期看见。WCN 需要把 Agent 从能力层推进到商业层，让它形成可计价、可配置、可复用的收入结构。" />
  <Card title="9.5 为什么 WCN 不依赖发币生存" description="这一页最重要的任务，是证明 WCN 即使没有 Token，也能先通过节点、服务、闭环与 Agent 建立自己的现金闭环。这样未来的 Token 才会建立在真实业务之上。" />
</CardGrid>
