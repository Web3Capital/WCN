## 资本配置层回答的核心问题：谁来接住需求？

项目带着需求进入 L1 之后，L2 的任务是把合适的资本方匹配过来。但「合适」不只是「有钱」，而是：

<CardGrid cols={2}>
  <Card title="赛道匹配" icon="target">资本方偏好的赛道（DeFi / AI / RWA / Infra / Gaming）是否与项目匹配。</Card>
  <Card title="阶段匹配" icon="chart-bar">种子 / Pre-A / A / 战略轮 / OTC——不同阶段的资本有完全不同的评估标准和决策周期。</Card>
  <Card title="结构匹配" icon="ruler">SAFE / SAFT / Token Warrant / Equity / Convertible Note——融资工具偏好是否兼容。</Card>
  <Card title="地区与合规匹配" icon="globe">资本方的 KYC/AML 要求、法域限制、地区偏好是否与项目兼容。</Card>
</CardGrid>

## 与 TradFi 资本配置的对比

<CardGrid cols={2}>
  <Card title="TradFi：Mandate-based" icon="landmark">GP 通过 LPA（Limited Partnership Agreement）获得明确的投资 Mandate——赛道、地区、阶段、票面都有约束。配置是系统化的。Blackrock 管理 $10T+ 资产，靠的不是个人关系，而是 Aladdin 系统的数据驱动配置。</Card>
  <Card title="Web3：Relationship-based" icon="smartphone">大多数 Crypto VC 的投资决策仍然高度依赖 GP 的个人关系和判断。没有标准化的偏好注册、没有系统化的匹配、项目到资本的路径不可预测。</Card>
</CardGrid>

> WCN 的 L2 不是要取代 VC 的判断（这是不可替代的人类能力），而是**把判断的前置环节系统化**——让合适的项目更快触达合适的资本，减少「找人介绍」的随机性。

## 进入系统的典型资本方

<CardGrid cols={2}>
  <Card title="Crypto VC" icon="coins">专注一级市场的加密原生基金。偏好早期、技术驱动、有 Token 经济设计的项目。票面通常 $100K-$5M。决策快（2-4 周），但门槛高。</Card>
  <Card title="Family Office / HNW" icon="landmark">更长期、更耐心的资金。偏好有真实收入或资产支撑的项目。票面 $500K-$10M+。决策慢（4-8 周），但关系深。</Card>
  <Card title="CVC / 战略资本" icon="handshake">交易所、公链、协议的战略投资部门。不只看财务回报，更看生态协同。能带来流量、技术合作和上架资源。</Card>
  <Card title="区域资本连接者" icon="globe">把特定地区的资金池（中东家族办公室、东南亚 HNW、欧洲机构资金）与全球项目连接起来。本地化能力是核心竞争力。</Card>
</CardGrid>

## WCN 的资本配置机制

<Steps>
  <Step title="偏好注册">资本方进入系统时注册投资偏好：赛道、阶段、票面范围、地区、融资工具、排除条件。偏好进入 Capital Profile。</Step>
  <Step title="智能匹配">当 L1 有新项目录入，系统基于 Capital Profile 自动推荐匹配度高的资本方。Research Agent 附上项目摘要和匹配理由。</Step>
  <Step title="结构化 Deal 准入">资本方查看结构化资料，决定是否进入 Deal Room。进入即意味着承诺投入时间和精力做进一步判断。</Step>
  <Step title="投资推进与反馈">资本方在 Deal Room 中推进尽调、提出问题、协商条款。所有交互被记录，用于后续归因和信誉累积。</Step>
</Steps>

<Callout type="tip">
资本配置层是 L1 和 L3 之间的关键枢纽——没有它，项目只是需求；有了它，需求才有机会进入 Deal 推进和结构化配置。WCN 的目标不是替代投资判断，而是让判断的效率提升一个数量级。
</Callout>
